# Gradient CSS generator

Générateur de dégradé CSS.

## Installation

Après avoir récupéré le repo, installez les packages Node :

```bash
yarn
```
ou
```bash
npm i 
```

Ensuite lancez l'application à l'aide de la commande :

```bash
yarn start
```
ou
```bash
npm start
```

## Stack

- Vanilla JS
- Webpack
- Babel
- Sass
